# TierTagger

TierTagger is a Minecraft tier-list utility. The source code is kept in a private repository; this public mirror exists so you can report issues, request features, and read the license.

---

## 🐛 Report a bug or request a feature

Please use the **[Issues tab](https://github.com/blackdmega-wq/tiertagger-public/issues)** of this repository.

When reporting a bug, please include:
- A clear description of what happened and what you expected.
- Steps to reproduce the issue.
- Screenshots or error messages if you have them.
- Your environment (OS, browser/Minecraft version, etc.) when relevant.
